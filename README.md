# Telegram Bot Minecraft v0.4.4
![GitHub release](https://img.shields.io/github/v/release/TRONMAXS/TelegramBotMinecraft?include_prereleases&label=release)
![GitHub repo size](https://img.shields.io/github/repo-size/TRONMAXS/TelegramBotMinecraft)
![GitHub issues](https://img.shields.io/github/issues/TRONMAXS/TelegramBotMinecraft)

**Telegram-бот для удалённого управления Minecraft-серверами через Telegram.**  
Позволяет включать, выключать, проверять статус серверов и выполнять команды через RCON.

## Возможности

* Управление неограниченным числом Minecraft-серверов
* Включение/выключение серверов
* Проверка статуса серверов и количества игроков
* Выполнение команд на сервере через Telegram
* Поддержка GUI на Windows (Windows Forms)
* Поддержка любых версий Minecraft-серверов (через RCON)

## Используемые технологии

* Язык: **C#**
* UI: **Windows Forms**
* Библиотеки: [`Telegram.Bot`](https://github.com/TelegramBots/Telegram.Bot), [`CoreRCON`](https://github.com/Challengermode/CoreRcon)

## Команды Telegram-бота

| Команда                                  | Описание                                                                        |
| ---------------------------------------- | --------------------------------------------------------------------------------|
| `/list_servers`                          | Показывает список доступных серверов                                            |
| `/select_server <password> <number>`     | Включает указанный сервер (по номеру в списке) — **требует пароль администратора** |
| `/start_server`                          | Запускает выбранный сервер                                                      |
| `/check_servers_status`                  | Проверяет статус всех серверов                                                  |
| `/players_count`                         | Показывает количество игроков на выбранном сервере                              |
| `/stop_server`                           | Останавливает выбранный сервер                                                  |
| `/server_command <password> <command>`   | Выполняет команду на выбранном сервере — **требует пароль администратора**         |
| `/delete_world`                          | Удаляет мир на выбранном сервере                                                |
| `/help`                                  | Отображает список доступных команд                                              |

## ⚙️ Конфигурация TelegramBotMinecraft

Бот использует **три конфигурационных файла**:

- `Settings.json` — настройки бота
- `Servers.json` — список серверов
- `UserSettings.json` — настройки пользователей

---

### 📁 `Settings.json`

```json
[
  {
    "Notifications": true,
    "BotToken": "123456789:ABCdefGHIjklMNOpqrSTUvwxYZ",
    "AdminPassword": "Admin",
    "ChatIds": [
      {
        "Identifier": "1234567890",
        "Name": "Admin"
      }
    ]
  }
]
```

#### Пояснение к полям:

- **Notifications** — включает (true) или отключает (false) всплывающие уведомления для этого приложения.

- **BotToken** — токен вашего Telegram-бота.

- **AdminPassword** — пароль администратора (используется для защищённых команд).

- **ChatIds** — список Telegram-чатов:

  - **Identifier** — **Telegram ID** чата или пользователя.

  - **Name** — отображаемое имя чата или пользователя.

### 📁 `Servers.json`

```json
[
  {
    "Name": "Vanilla(Survival) - 1.20.1",
    "StartupPath": "G:\\MinecraftServers\\Survival\\start.bat",
    "FolderPath": "G:\\MinecraftServers\\Survival\\",
    "Ip": "127.0.0.1",
    "RconPort": "25575",
    "RconPassword": "12345",
    "ConnectIp": "127.0.0.1",
    "Port": "25565"
  }
]
```

#### Пояснение к полям:

- **Name** — имя сервера, отображается в списках и GUI.

- **StartupPath** — путь к **.bat** или **.cmd** для запуска сервера.

- **FolderPath** — путь к папке сервера.

- **Ip** — IP сервера для RCON-соединения **(обычно 127.0.0.1, если бот и сервер распологаются в одной сети)**.

- **RconPort** — порт для подключения по RCON **(по умолчанию 25575)**.

- **RconPassword** — пароль для RCON-доступа.

- **ConnectIp** — IP, по которому игроки подключаются к серверу.

- **Port** — порт Minecraft-сервера **(по умолчанию 25565)**.

### 📁 `UserSettings.json`

```json
[
  {
    "Identifier": "1234567890",
    "Name": "Admin",
    "AllowedServers": [
      {
        "Server": "Vanilla(Survival) - 1.20.1",
        "Enabled": true
      }
    ],
    "AllowedCommands": [
      {
        "Command": "/help"
      },
      {
        "Command": "/start"
      }
    ]
  }
]
```

- **Identifier** — **Telegram ID** пользователя из Settings.json.

- **Name** — отображаемое имя пользователя из Settings.json.

- **AllowedServers** — список серверов, к которым пользователь имеет доступ:

  - **Server** — имя сервера из Servers.json.

  - **Enabled**  — выбран ли сервер для пользователя:  
            **true** — выбран (будет использоваться при выполнеии команд этим пользователем),  
           **false** — не выбран (будет игнорироваться).

- **AllowedCommands** — список доступных команд для пользователя:

  - **Command** — команда, которую можно использовать (например, /start, /help).



## 🗂 Резервное копирование
Теперь при закрытии приложения все три JSON-файла автоматически сохраняются в:

- **ServersBackup.json**

- **SettingsBackup.json**

- **UserSettingsBackup.json**

Это нужно для того, чтобы при возникновении ошибок в основных файлах (Servers.json, Settings.json, UserSettings.json) можно было быстро откатиться к последним корректным значениям.  
**Бэкап-файлы перезаписываются при каждом закрытии приложения**.

## 🧩 Установка и запуск

### 📦 Установка

1. Перейдите на вкладку [Releases](https://github.com/TRONMAXS/TelegramBotMinecraft/releases) и скачайте архив с установщиком.
2. Распакуйте архив в любое удобное место.
3. Запустите `SetupTelegramBotMinecraft.msi`.
4. Следуйте инструкциям установщика.

---

### 🚀 Запуск

После установки:

1. Найдите ярлык **"TelegramBotMinecraft"** в меню Пуск или на рабочем столе.
2. Запустите программу.
3. При первом запуске выполните настройку:

#### 🔧 Настройка бота:

- После запуска программа сворачивается в трей (рядом с часами).
- Кликните правой кнопкой мыши по иконке в трее и выберите **"Открыть"**.
- Перейдите во вкладку **"Настройки"**:
  - Введите **токен вашего бота** и нажмите **"Обновить токен"**.
  - Добавьте пользователей:
    - Укажите имя (для отображения) и **Telegram ID**.
    - Нажмите **"Добавить"** для сохранения.
    - Чтобы удалить пользователя — выделите его и нажмите **"Удалить"**.
- Закройте окно настроек.
- Нажмите кнопку **"Переподключиться к боту"**.

#### 🌐 Настройка серверов:

- Нажмите кнопку **"Сервера"**.
- В открывшемся окне:
  - Отредактируйте пример вручную или нажмите **"Открыть"**, чтобы загрузить готовый файл с настройками.
  - После редактирования нажмите **"Сохранить"**.
  - Закройте окно.
- Нажмите кнопку **"Перепроверить сервера"**, чтобы применить изменения.

4. ✅ Бот готов к работе.

---

> ⚠️ Если приложение не запускается — скорее всего, требуется [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/ru-ru/download/dotnet?cid=getdotnetcore).  
> Установите его и перезапустите программу.

> 💡 Программа **не добавляется в автозагрузку автоматически**. При необходимости добавьте её вручную.

## Дополнительно

* Ограничений на количество серверов — **нет**

  > 🔐 Пароль можно изменить в настройках приложения.
